《番茄工作法图解》&《单核工作法图解》

Posted on 2018-10-30 | In 读书笔记（自我管理类） | Visitors:

一次只做一件事

步骤

  1. 列出待办的活动
  2. 选出任务中最重要的，最想尽快完成的一项
  3. 开启番茄钟
  4. 全神贯注的做这一项任务
  5. 到闹钟响，立即停下，别管人物有没有完成
  6. 休息 五分钟后继续

原理

  • 面对复杂，望而却步
    • 那些复杂的、在一个 25 分钟内完不成的任务，往往会造成拖延
    • 拖延使我们苟且偷安，然而难题并没有消失
    • 别考虑任务有多复杂，重要的是开始、开始
  • 不要想着完成整件事情好需要多少工夫，而是想想要完成这个番茄钟其实也没多久
  • 番茄工作法要求在每天早晨做计划，量力而为的选择当天能完成的活动，列入“今日待办”表格。然后在每个番茄钟之前，重新评估活动的优先级，确保自己一直在做最重要的事情，而不是别的事情
  • 将打开番茄钟，遵守铃声，填写待办表格的动作变成习惯，充分利用条件反射的力量
  • 番茄工作法要求在一天结束前要做三件事：记录、处理和可视化
    • 将活动拆分成小项目：预估某项活动需要的番茄数
    • 定量预估：通过比较“预估番茄数”和“实际完成所用的番茄数”，得到及时反馈
    • 定性预估：将活动进行过程中冒出的一些次要任务或想法填入“计划外紧急”一栏，然后再接再厉，专心致志的完成当前的主要活动
    • 跟踪全天工作流程的真实数据，用于每日回顾，以改进第二题的流程
    • 可以从计算“中断数”和“完成番茄数”开始
  • 问自己：一次只做一件事
    • 什么让你半途而废
    • 在哪些地方读书，更容易集中精力
    • 怎样逃避做无聊琐事
    • 通常哪类活动花费的时间超出预期
    • 时间是否经常被事务型活动填满，即使有更重要的事情要办

背景

大脑结构

  1. 大脑的结构可以分为四个层次：脑干、边缘系统、皮层、额叶
    1. 脑干：爬行动物也有，提供了可靠的条件反射机制，不需要参与意识即可作出反应
    2. 边缘系统：人类和哺乳动物共有的。帮助我们进行长期记忆，负责处理奖励信号，功能主要涉及情感和社会关系
    3. 大脑皮层：完善程度和动物的智能成正比，使我们能够认知和思考
    4. 额叶：使人类和其它动物不同，让我们异想天开，发明创新，据理力争，理解异见，计划协商，操纵大局

生物节律

  1. 脑干负责控制多种基本节律。有规律有节奏的生活方式，最适合我们的大脑
  2. 节律的变更会导致头脑混乱
  3. 作息时间不规律的人往往更容易迷茫和焦虑

左脑与右脑

  1. 大脑皮层涵盖两个大脑半球，通常称为左脑和右脑
  2. 两个半球在主要智力活动上各有分工，但都具备所有领域的基本技能
  3. 左脑倾向于匹配，对我们当前看到的和之前经历过的进行比较
  4. 右脑更倾向于直觉，理解比喻、产生梦想、组合新想法，带来灵光一现

多动

  1. 如果感到难以在一项单一活动上全神贯注较长时间，那应该属于典型的多动症
  2. 当感到非常疲惫时，可能会显出多动型人群的类似症状：后继乏力，精神不集中。因此，需要休息以获得新能量
  3. 要让注意力处于最佳状态，可以在每半小时之间进行短暂休息，每周不安排超过 40 小时的工作计划，采取可持续的步伐

工作记忆

  1. 阿兰·巴德利描述了大脑中用于暂存和处理信息的结构和程序
    1. 中央执行器：控制认知过程，对信息进行分类
    2. 语音回路：通过复述的方式存储我们看到的东西，容量有限，使得短单词更容易记忆
    3. 视空画板：保存了我们所看到的信息，帮助判断距离或进行视觉想象
    4. 情节缓存：跨领域进行沟通，建立起同一的视觉、空间、语言和时序单元，帮助长期记忆
  2. 工作记忆容量有限，环境的频繁变化会降低生产力。因此大部分人不能分心同时处理两个问题，如果在忙于一项任务时，注意力转移到其他事情，等回到刚才在做的事情，就得多花时间让自己回到轨道上来

联想机器

  1. 阿兰·巴德利在 1966 年提出，根据她对记忆编码方式的研究，在短期记忆中信息通常以声音形式存储
  2. 长期信息通常以语义形式存储，就像一台巨大的联想机器
  3. 大脑通过联想对记忆进行编码，可视化技术，比如思维导图，是理清脉络、服务长期记忆的极佳工具

交替和持续

如果我们能将工作视为一连串的事件各个击破，生产力将得以提高

做梦

  1. 在快速眼动睡眠期间，大脑的一小部分重放我们日常生活的影响，这部分被称为海马体
  2. 记忆是一份稳定封存的副本，与事实无关的情绪化成分均被剔除
  3. 大脑还会在所收集的各组记忆之间找出新的联系
  4. “脱机状态”——即大脑没有思考问题的时候，能为我们提供新思路和见解

吸收

  1. “相关记忆”能够综合概况个人原有知识，脱机状态对他意义重大，包括快速眼动睡眠时间
  2. 脱机状态后，对相关度的认识可以获得提高

吃饭睡觉钟

心流

  1. 心流是一种精神状态，具有以下特点
    1. 明确的目标
    2. 集中
    3. 专注
    4. 自我意识消失
    5. 时间感扭曲
    6. 直接
    7. 即时反馈
    8. 能力水平与面对挑战的平衡
    9. 个人控制感、工作本身的内在奖励、行为和认知的合一
  2. 心流是一种创造性的状态，但是缺乏全局观念，创造力与统筹力难以并存
  3. 在进入心流状态之前，我开启一个番茄钟，待他唤醒我后，我可以暂时换上战略眼光，观察全局，然后再次回到心流中。这就是节奏

激励

  1. 刚刚开始一项重要活动，需要收集、分析大量的信息，面前有千头万绪，每条信息都是独特的，没有重复和冗余
  2. 随着时间推移，从起初的一团乱麻中，我开始看到脉络
  3. 我学到的越多，任务变得越有趣
  4. 某些环节上，我甚至完全进入状态，满脑子都是它，欲罢不能
  5. 然后，我看透了整个事情后，他就变得越来越乏味
  6. 最重要的部分已完成，但仍然有收尾工作
  7. 激励是一种生理活动加剧的状态，我不需要巅峰，更不想要低估，我想保持一个可持续的步伐

拖延

  1. 拖延是一种焦虑应对机制，与启动或完成某项任务或决策有关
  2. 有三个导致拖延的明显原因
    1. 其他人强迫你做一些事，违背你的意志
    2. 你给自己压力，想要有完美的表现
    3. 害怕犯错误或受批评
  3. 拖延会暂时缓解压力。但压力来自内部，而治疗的良方是找到项目的起跑线

英雄主义和内疚

长远而言，如果能有一个可持续的步伐，完美的生产率会更好。完美的大脑需要定期的休息，既包括工作时间的短暂休息，也包括每天工作完成后的较长休息

薄片撷取

  1. 正确的决策是意识和直觉之间的平衡。为了做出正确的绝对，你需要简化问题，使其只包含重要事实和你有足够能力进行分析的领域
  2. 薄片撷取是一种不假思索的思考方式。我们根据一段极短时间内的经验，从中提取真正重要的信息。这些判断有效与否，完全取决于我们的经验、专业训练和知识，而非其他因素。
  3. 真正的挑战在于建立一种简单、透明、具支持性、自我完善、有灵活性的进程
  4. 灵活性是基于我们跟踪记录的事实数据，不仅要用这些指标调整进程本身而且要改善我们自己
  5. 然后，我们能够在正确的时间，以正确的方式作出瞬间决定，并据此采取正确的行动

拥抱变化

  1. 变化是商务世界永恒的主题，学会拥抱变化，打开机会之窗，消解压力
  2. 变化的代价是成本暴增。基于这一理由，我们必须要事先认真计划，以避免后期的大幅修改
  3. 但即使变化发生了，所增加的成本通常也可以借助良好的对策加以抵消：严格控制我们必须完成的内容，并经常性地重新评估，从长期和短期角度问自己：“完成什么才是 最重要的？”
  4. 延缓与变化相关的成本产生，为回到正轨争取最大的机会

选择的悖论

  1. 选择产生焦虑，评估可选方案，会导致时间流失和专注度减少
  2. 我们自始至终愿意做最重要的事情，当环境发生改变，而且影响到我们手头工作的价值时，我们应该接受这种变化，或许还要调整原先的优先次序
  3. 即使在万不得已的情况下，重新调整优先次序，即“选择当前活动”，也必须限制在某种场合下才能进行
  4. 但为了及时对变化作出响应，即“切换当前活动”，我们必须让此类场合足够频繁地出现
  5. 因此需要一个步调，例如选择—工作—休息；选择—工作—休息……同时我们用时间盒限定工作时段，以便我们全神贯注于当前工作的时候，还能抽出身来

问自己：背景

  • 在一份耗费脑力的活动上你能专注多久？
  • 你的私人生活和职业生活的节奏是怎样的？
  • 对于考验记忆的游戏，你能玩的好吗？
  • 你上一次拖延是什么时候？为什么？
  • 你做决定的时候是凭直觉还是凭意识？如何平衡？
