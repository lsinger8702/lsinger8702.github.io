Git - 分布式版本控制系统

  • 不需要联网
  • 安全：每个人电脑里都有完整的版本库，某一个人的电脑坏掉了不要紧，随便从其他人那里复制一个就可以了
  • 强大的分支管理

常用命令

  • 创建版本库 & 提交文件

    git add 把工作区（Working Directory）的文件添加到暂存区（Stage）

    git commit 把暂存区的所有修改提交到分支

    提交之后，如果工作区没有被修改过，那工作区就是 ”clean“，暂存区是空的

    暂存区和分支属于 Git 版本库，工作区不属于版本库

  • 修改文件 & 提交

    Git 提交的是修改而非文件，也就是说，没有被 add 到暂存区的文件无法被提交到分支

  • 回退修改

    HEAD 为当前版本，HEAD^^ 为上个版本，head^^ 为上上个版本，HEAD~100 为前 100 个版本

  • 丢弃修改

  • 删除文件
